xcorso

Orienteering course planner that runs anywhere in your browser, Mac, Linux, Windows, and tablets. Load an OCAD, PDF, or image map and start placing controls. When you are done, export IOF XML and print-ready PDFs. Free and open source.

Try this demo here!

Any map format

OCAD files, PDFs, and bitmap images. Accurate scale detection for OCAD; manual calibration for the rest.

Full course design

Linear and Score-O courses, butterfly loops, map exchanges, taped routes, funnels, and control descriptions.

Print layout

WYSIWYG page editor with overprint, scale bars, clue sheets, and multi-course output.

IOF XML export

IOF XML v3 for race-day use with SPORTident, or any IOF-compatible timing system.

Touch-ready

Full touch support. Design courses on an iPad or any other tablet in the field.

Cloud sync

Sign in to sync projects across devices. Or use it fully offline — no account required.
